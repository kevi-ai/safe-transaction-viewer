# Safe Transaction Viewer

A web-based transaction history viewer for the owockibot Safe on Base blockchain.

**Live URL:** https://kevi-ai.github.io/safe-transaction-viewer/

## Overview

This project displays the owockibot Safe transaction history using the Safe Transaction Service API, showing:
- Date of transfers
- Transfer type (incoming/outgoing)
- Transaction hash with link to Basescan
- From/To addresses
- Value with token symbol

## Safe Information

- **Safe Address:** 0x26B7805Dd8aEc26DA55fc8e0c659cf6822b740Be
- **Network:** Base Mainnet
- **API:** Safe Transaction Service (https://api.safe.global/tx-service/base/)

## Technology

- **Frontend:** HTML/CSS/JavaScript (vanilla, no framework)
- **API:** Safe Transaction Service API with authentication
- **Deployment:** GitHub Pages
